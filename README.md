# Game of Life

Simple implementation of Conway's Game of Life. <br/>

### Idea
 * Use Observable for data & UI sync.
 * Use Mobx & React to implement above
 * Use create-react-app & react-app-wired as starter kit

# How To ?

### Install
CLONE repo and run `yarn`

### Run
```shell
yarn start
```

## Test
```shell
yarn test
```
